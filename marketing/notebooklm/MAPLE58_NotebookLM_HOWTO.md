# MAPLE58 → NotebookLM Video Overview (slideshow) — How To

NotebookLM generates the narrated slideshow ("Video Overview") inside Google's web app.
There is no API and it needs your Google login, so the generate click is yours. Source
material below is done — this is the 4-step run.

## Steps

1. Go to **notebooklm.google.com** and sign in (respectfulnrespected59@gmail.com).
2. **New notebook** → **Add source** → upload **`MAPLE58_NotebookLM_Source.md`**
   (this folder). Optional second source: paste the YouTube link
   `https://youtu.be/vxzFZG1xtJk` so it can pull from the existing explainer too.
3. In **Studio** (right panel) → **Video Overview** → click **Customize** and paste the
   FOCUS PROMPT below → **Generate**. (Takes a few minutes; it renders narrated slides.)
4. Download the video → it becomes the companion slideshow. (Optionally also generate
   the **Audio Overview** = a 2-host Deep Dive podcast from the same source — great as a
   Gumroad value-add.)

## FOCUS PROMPT (paste into Customize)

```
Audience: complete beginners to options trading who have been burned or talked down to.
Tone: honest, plain-English, no hype, encouraging but never promising riches. Repeat the
core truth that most options trades lose and survival is the real skill.

Structure the slideshow as four acts — GET, KEEP, GROW, SHARE — plus a short intro
(what MAPLE58 is and why it won't lie to you) and a closing slide ("slow and alive beats
fast and liquidated; educational, not financial advice").

Must explain in simple terms: the nudge (a lean, not an order), the TTM squeeze, the
probe (a small first bet), the delta 0.25 readability floor (why the cheapest option is a
dead lottery ticket), the 20%-of-pot rule and "walk," debit spreads vs strangles,
Probe -> Read -> Escalate, the Pot Tracker ($300 -> $333), and Proof Mode (shows wins AND
losses net of fees). Always show option prices as real per-contract dollars (premium x 100).

Keep it concrete and visual. Do NOT give buy/sell recommendations or imply guaranteed
gains. End every directional idea with: the choice is yours.
```
